Slices

Slices其实是和数组紧密相连的，slices提供了一种更轻量级的方式来访问数组的元素，slices主要由三部分构成

  • pointer // pointer指向数组中的一个元素
  • length　 // slices里面元素的个数
  • capacity // slices开始的位置到整个他指向的数组结束的位置.
